¿Qué es Appunto?

Appunto es una API REST desarrollada en JavaScript, utilizando Express, Mongosh y Mongoose para una integración sólida con una base de datos MongoDB.
Emplea Bcrypt y JSON Web Token para garantizar la seguridad de los datos además de Nodemailer, para confirmar los correos electrónicos.

¿Para qué sirve?

Appunto conecta profesores y alumnos para facilitar la resolución de dudas y fomentar el aprendizaje colaborativo.
Los alumnos pueden apuntarse a una lista de espera para recibir asistencia y también ayudar a sus compañeros al responder preguntas.
Los profesores obtienen métricas sobre el desempeño, la interacción de los alumnos y otorgan puntos positivos que fomentan la motivación.

¿Cómo utilizar Appunto?

Todos los usuarios deben registrarse en la plataforma.
Para acceder a la plataforma hay que aceptar el correo de confirmación.
Cuando un alumno necesita ayuda, puede enviar una pregunta que será visible para el profesor y otros compañeros.
